A flash loan protocol on the Stacks blockchain, enabling uncollateralized borrowing within a single transaction block.

## Features
- **Flash Loans:** Borrow assets without collateral if returned in the same tx.
- **Liquidity Pools:** Provide liquidity to earn fees.
- **Arbitrage:** Enable cross-DEX arbitrage opportunities.

## Structure
- `frontend/`: Next.js dashboard.
- `smartcontract/`: Clarity contracts.
